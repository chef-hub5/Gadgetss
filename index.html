Google Apps Script Setup Required

To enable contact form and order management with email notifications, follow these steps:

  1. Go to script.google.com
  2. Create a new project and paste the provided Google Apps Script code
  3. Deploy as web app with execute permissions for "Anyone"
  4. Copy the deployment URL and replace YOUR_SCRIPT_ID in the code
  5. The script will automatically create Google Sheets for contacts and orders
  6. Payment receipts will be saved to Google Drive
  7. Email notifications will be sent automatically
